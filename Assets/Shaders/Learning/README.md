# Shader Learning Workspace

This folder is the dedicated workspace for the shader learning track.

Structure:

- `01_Basics`: first-stage exercises and notes
- `Week01_Basics`: the complete week-one study set with a finished demo scene
- `Week02_StylizedToon`: ramp lighting, outline, rim light, toon specular, MatCap, shadow pattern, and a final advanced toon product
- `02_Toon`: toon and stylized rendering exercises
- `03_Surface`: texture, normal, transparency, and lighting variants
- `04_Post`: post-processing and screen-space effects
- `05_SRP`: URP/SRP practice when we get there

The rule is simple:

1. Read one example from the book project.
2. Copy it into this folder.
3. Modify it into your own version.
4. Keep each lesson as a separate shader so the progress is visible.
